# interlocking-flat-surface-mechanism
A 3D printable interlocking flat surface mechanism

## Overview
This project is a simple mechanical cube/block design made for 3D printing.

## Files
- `stl/` → printable STL files
- `images/` → photos of the mechanism
- `video/` → demo video showing movement/function

## Description
This mechanism is designed as an interlocking flat-surface-style assembly.
It basically works by having two rotating magnets(one surface each) automatically locking to each other.
The magnet used for this design is a 15x5 mm disk ferromagnet.
The two parts of the surface can be glued or nailed together.

## Printing
Print the STL files using your preferred slicer settings.
